This project is an Intermediate HTML, CSS, and JavaScript application developed to enhance web development skills by implementing a Contact Form, JavaScript Form Validation, Responsive Layout, and a Dynamic To-Do List. The contact form allows users to enter their name, email, and message, while JavaScript validation ensures that all required fields are completed and the email format is correct before submission. The webpage uses Flexbox for navigation and CSS Grid for content organization, making it fully responsive across desktop, tablet, and mobile devices. Additionally, a dynamic to-do list is implemented using JavaScript DOM manipulation, enabling users to add and remove tasks in real time. This project demonstrates practical usage of HTML5, CSS3, and JavaScript to create interactive, responsive, and user-friendly web applications.
